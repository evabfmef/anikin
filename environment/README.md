# Environment
 
This directory documents the downstream software environment used for analysis and figure generation in the anikin project (Stare et al., 2026).
 
## Contents
 
- `requirements.txt` — Python packages and versions used in the analysis and figure-generation scripts
- `R_packages.txt` — R packages and versions used in the analysis and visualization workflows

## Notes
 
The package lists were compiled from the analysis environment used during the study and are provided for transparency. They are not intended to function as lockfiles or exact reproducible environment specifications.
 
Upstream analyses additionally relied on external bioinformatics tools, which are summarized together with their versions in `/docs/software_versions.md`.
